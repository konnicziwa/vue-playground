# Webstorm Keymap

- `ctrl + shift + p` Prettier
- `ctrl + shift + d` Duplicate line
- `ctrl + shift + w` Find selection for the next occurrence
- `ctrl + d` Remove line
- `alt + up` Move line up
- `alt + down` Move line down
- `ctrl + enter` Start new line
- `ctrl + alt + n` New file / directory
- `f2` Rename
